# ShellyControlBroker
 
Version 0.1.2
This application is used to control Shelly4Pro relays with the following tasks;
- Implementation of a MQTT broker
- Evaluate the MQTT messages
- manage the MQTT client (Shelly relay) connected to the broker
- Evaluate the load of the connected Shelly relays
- Control the connected Shelly relays (switch on/off etc.)
The application is in the process of development.
